Que es Bootstrap

Bootstrap es el framework de HTML, CSS y JavaScript más popular del mundo, empleado para el desarrollo de interfaces Web, adaptando la pantalla de la página web al tamaño de la pantalla de cualquier dispositivo desde el que se vaya abrir. En términos más técnicos es un conjunto de estilos, efectos, configuraciones responsivas, componentes, partes y piezas que podemos utilizar a nuestro favor al momento de construir un sitio web.

La última versión de Bootstrap es la número 4, en la cual se ha incorporado nuevas funciones respecto a la versión anterior, como lo son los paneles contenedores para tarjetas, nuevas unidades de medida para la tipografía (rems). En esta nueva versión también se añade una nueva clase llamada outline para los botones mostrando solo un borde, además de haber incorporado los plugins de JavaScript (ECMAScript 6).

Que es Sass

Sass(Syntactically Awesome StyleSheets) Es un preprocesador de hojas de estilos CSS, es una nueva forma de escribir nuestras CSS introduciendo en ella conceptos de la programación, como variables, funciones y otros conceptos pertenecientes a las hojas de estilos que Sass mejora considerablemente, intentando alcanzar la filosofía Rails como: No te repitas tener nuestros proyectos mejor organizados, ahorrar código y tiempo, la cual podemos realizar de forma rápida reutilizando código, entre otros.

La gran cualidad de SASS es que te permite llevar tu CSS un paso más adelante, ya que entre otras cosas, permite reutilizar código, organizar las clases y permite trabajar mucho más rápido que con el CSS de forma tradicional.

Instalacion y uso

1. Para utilizar Sass antes debemos instalar Ruby.

2. Después de instalar los requisitos abrimos la consola de ruby.

3. Estando en la consola de ruby ejecutamos el siguiente comando gem install sass.

4. En la carpeta css de nuestro proyecto agregamos un archivo con extensión .sass(main.sass).

5. Ya podremos darle estilos a nuestra página utilizando Sass, como se muestra en el siguiente video.
